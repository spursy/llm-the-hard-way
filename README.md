## install reference

```
npm install --dev 

npm install -g git+https://github.com/streetsidesoftware/cspell-cli
```

## 贡献指南

- 新增论文解读（添加至 deep-day/）
- 优化代码示例（transformer/目录）
- 完善术语解释（knowledge/目录）
